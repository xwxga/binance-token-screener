## 🎯 版本概述

币安代币筛选器 v1.0 是一个完全重构的生产级别代币分析工具，专为专业交易者和分析师设计。本版本完全实现了 md.txt 中的所有需求，解决了所有技术问题，并新增了智能文件管理系统。

## ✨ 核心功能

### 🔐 认证与稳定性
- **OAuth认证**: 使用个人谷歌账户(15GB存储)解决403错误
- **用户可配置数量**: 可自定义现货和期货代币数量(默认各80个)
- **14日历史分析**: 平均交易量和累计收益分析
- **实时资金费率**: 实时期货合约资金费率和持仓量数据
- **精细化异常检测**: 单元格级别高亮显示，详细标注逻辑
- **代币过滤**: 排除已下架/重命名代币(ALPACA, BNX, LINA, VIDT, AGIX, FTM)
- **每日独立文件**: 不覆盖历史记录，每天创建新分析

### 📊 数据分析功能
- **8个增强工作表**: 原始数据、交易量排行、市值排行、涨跌排行、期货专注、每日涨幅榜、低量高市值、低市值高交易量
- **17+数据列**: 完整市场分析，包括价格、交易量、收益、资金费率、持仓量数据、异常标注
- **精细化单元格高亮**: 8种不同颜色代码标识特定异常类型
- **智能过滤**: 在异常检测中排除主要币种(BTC, ETH, SOL, XRP)
- **特殊合约处理**: 正确映射1000x合约(PEPE, BONK, FLOKI, SHIB)
- **标注逻辑**: 详细说明每个异常检测的原因

### 📁 智能文件管理系统 (v1.0新增)
- **自动文件夹创建**: 为每次分析创建有序的文件夹结构
- **文件类型分类**: 将CSV、Excel、报告和日志分别保存到专用文件夹
- **按日期组织**: 每天创建新的时间戳文件夹
- **清洁项目结构**: 不再有散落在项目根目录的文件
- **便于备份和分享**: 完整的分析结果在一个有序文件夹中

**文件夹结构**:
```
币安代币分析结果_YYYYMMDD/
├── CSV文件/          # 各个工作表的CSV文件
├── Excel文件/        # 合并的Excel工作簿
├── 分析报告/         # 分析报告和总结
└── 日志文件/         # 日志和诊断文件
```

## 🚀 安装与设置

### 环境要求
```bash
# 必需的Python包
pip install pandas requests gspread google-auth google-auth-oauthlib google-auth-httplib2 google-api-python-client openpyxl
```

### OAuth设置 (一次性)

1. **获取OAuth凭据**:
   - 访问 [Google Cloud Console](https://console.cloud.google.com)
   - 创建或选择项目
   - 启用Google Sheets API和Google Drive API
   - 创建OAuth 2.0客户端ID(桌面应用程序)
   - 下载凭据文件并重命名为 `oauth_credentials.json`

2. **添加测试用户** (重要):
   - 在Google Cloud Console中，转到"OAuth同意屏幕"
   - 在"测试用户"部分添加您的Gmail地址

3. **运行OAuth设置**:
```bash
python oauth_setup_v1.0.py
```

## 📋 工作表详情

### 1. 原始数据 (原始数据)
包含所有17+列的完整数据集，包括价格、交易量、收益、资金费率、持仓量数据。

### 2. 交易量排行 (交易量排行)
按24小时总交易量排序的代币，包含精细化异常检测和标注逻辑。

### 3. 市值排行 (市值排行)
按市值排序的代币，包含交易量相关性和异常高亮显示。

### 4. 涨跌排行 (涨跌排行)
按1日收益排序的表现最佳和最差代币，包含资金费率异常检测。

### 5. 期货专注 (期货专注)
期货特定分析，包括资金费率、持仓量数据、7日平均持仓量和交易量比较。

### 6. 每日涨幅榜 (每日涨幅榜)
每日涨幅前列代币的历史跟踪，包含标注逻辑。

### 7. 低量高市值 (低量高市值)
高市值但低交易量的代币，潜在价值发现机会。

### 8. 低市值高交易量 (低市值高交易量)
低市值但高交易量的代币，潜在投机机会。

## 🎨 异常检测与颜色编码

### 异常类型
- **期货量增长**: 期货交易量显著增长 (浅红色)
- **现货量增长**: 现货交易量显著增长 (浅橙色)
- **1日涨跌前5**: 1日涨跌幅前5名 (浅黄色)
- **14日涨跌前5**: 14日涨跌幅前5名 (浅绿色)
- **量/市值极高**: 交易量与市值比例极高 (浅蓝色)
- **资金费率异常**: 资金费率异常高或低 (浅紫色)
- **持仓量异常**: 持仓量异常变化 (浅青色)

### 智能过滤
- 自动排除主要币种(BTC, ETH, SOL, XRP)的异常检测
- 专注于中小市值代币的异常发现
- 提供详细的标注逻辑说明

## 🔧 故障排除

### 常见问题
1. **403错误**: 确保使用OAuth认证而非服务账户
2. **网络连接**: 使用离线模式生成本地文件
3. **权限问题**: 确保Gmail地址已添加为测试用户
4. **文件权限**: 确保有写入权限创建输出文件夹

### 网络诊断
程序包含内置网络诊断功能，可自动检测连接问题并提供解决方案。
