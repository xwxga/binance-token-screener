# 🎯 增强版币安代币筛选器 v2.0 使用指南

## 📋 概述

增强版币安代币筛选器是一个功能强大的加密货币数据分析工具，提供实时市场数据获取、智能异常检测、历史数据追踪等功能。

## 🆕 v2.0 新增功能

### 1. 数据库历史追踪
- **SQLite数据库**: 自动保存每日数据到本地数据库
- **历史记录**: 支持查询任意代币的历史表现
- **每日涨幅榜**: 记录每日涨幅前20名的历史数据

### 2. 新增标签页
- **每日涨幅榜**: 显示历史涨幅记录和趋势
- **低市值高交易量**: 发现潜在投机机会的代币

### 3. 异常值检测系统
- **自动检测**: 识别交易量、涨跌幅、资金费率等异常值
- **智能标注**: 用醒目标记突出异常数据
- **分级警告**: 高/中/低三级异常严重性分类

### 4. 每日独立文件系统
- **每日文件**: 每天生成独立的Google Sheets文件，格式为 "Binance_Token_screener_Analysis_YYYY-MM-DD"
- **历史保护**: 不覆盖历史文件，所有历史数据永久保留
- **智能检测**: 自动检测和显示过去7天的历史文件链接
- **快速访问**: 程序输出中提供历史文件的直接链接

### 5. 数据展示优化
- **无限制**: 移除固定50条数据限制，显示所有可用数据
- **完整表头**: 补充所有缺失的指标列
- **性能优化**: 大数据集分批上传，提升加载速度

## 📊 标签页详解

### 1. 原始数据
- **用途**: 显示所有代币的完整数据
- **排序**: 按交易量排序
- **包含**: 价格、市值、交易量、持仓量、涨跌幅等

### 2. 交易量排行
- **用途**: 按24h总交易量排序
- **异常检测**: 
  - 期货交易量超过14日均值的代币
  - 1日涨跌幅前5名
  - 14日涨跌幅异常值

### 3. 市值排行
- **用途**: 按市值大小排序
- **异常检测**:
  - 高市值低交易量 (交易量/市值 < 1%)
  - 高市值高交易量 (交易量/市值 > 100%)
  - 涨跌幅异常 (1日涨跌 > 15%)

### 4. 涨跌排行
- **用途**: 显示涨幅榜和跌幅榜各50名
- **异常检测**: 交易量超过90分位数的代币

### 5. 期货专注
- **用途**: 专注期货市场，排除主流币(BTC/ETH/SOL/XRP/BNB)
- **异常检测**: 资金费率绝对值超过1%的代币

### 6. 低量高市值
- **用途**: 发现市值高但交易量低的代币
- **筛选条件**: 市值>50分位数，交易量<30分位数

### 7. 每日涨幅榜 🆕
- **用途**: 显示历史7天的每日涨幅前20名
- **价值**: 识别持续表现强劲的代币

### 8. 低市值高交易量 🆕
- **用途**: 发现市值低但交易活跃的代币
- **筛选条件**: 市值<50分位数，交易量>70分位数
- **风险提示**: 投机性较强，风险较高

## 🚨 异常值类型说明

### 交易量异常
- **期货交易量异常**: 24h期货量超过14日均值
- **交易量激增**: 超过90分位数的异常交易量

### 涨跌异常
- **1日涨幅异常**: 单日涨幅超过10%
- **14日涨跌异常**: 14日涨跌超过50%
- **涨跌幅异常**: 1日涨跌超过15%

### 期货市场异常
- **资金费率异常**: 绝对值超过1%
- **极端资金费率**: 绝对值超过2%

### 市值交易量异常
- **高市值低交易量**: 交易量/市值比率低于1%
- **高市值高交易量**: 交易量/市值比率超过100%

## 📈 指标解释

### 基础指标
- **价格(USDT)**: 当前USDT价格
- **市值(M USD)**: 流通市值，百万美元
- **24h总量(M USDT)**: 现货+期货24小时总交易量
- **24h现货交易量**: 现货市场24小时交易量
- **24h合约交易量**: 期货合约24小时交易量

### 历史指标
- **14日平均现货交易量**: 过去14天现货交易量均值
- **14日期货合约交易量**: 过去14天期货交易量均值
- **持仓量(M USD)**: 期货合约当前持仓量
- **7日平均持仓**: 过去7天持仓量均值

### 收益率指标
- **1日涨跌**: 过去24小时价格变化百分比
- **3日涨跌**: 过去3天价格变化百分比
- **7日涨跌**: 过去7天价格变化百分比
- **14日涨跌**: 过去14天价格变化百分比

### 期货指标
- **资金费率**: 期货合约资金费率
- **交易量/市值**: 交易活跃度指标
