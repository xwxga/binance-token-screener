# 🕐 币安代币筛选器简单定时运行系统

## 🎯 功能概述

这是一个简化版的自动定时运行系统，**无需系统权限**，只需要在后台持续运行一个Python脚本即可。

### ✅ 主要特点

- ⏰ **每天7:45自动执行**：精确定时运行
- 🔒 **无需系统权限**：不需要macOS系统权限
- 🌐 **支持网络代理**：自动读取配置文件中的代理设置
- 📝 **完整日志记录**：详细的运行日志
- 🔄 **简单易用**：一键启动、停止、查看状态
- 💻 **后台运行**：关闭终端窗口后继续运行

## 📁 文件说明

- `simple_scheduler.py` - 简单定时调度器主程序
- `start_simple.sh` - 启动管理脚本
- `scheduler_config.json` - 配置文件（可选，用于代理设置）

## 🚀 使用方法

### 1. 启动调度器

#### 方法1：后台运行（推荐）
```bash
./start_simple.sh background
```
- 程序在后台运行
- 关闭终端窗口后继续运行
- 每天7:45自动执行

#### 方法2：前台运行（用于测试）
```bash
./start_simple.sh start
```
- 可以看到实时输出
- 可以立即测试运行
- 关闭终端窗口程序会停止

### 2. 管理命令

```bash
# 查看运行状态
./start_simple.sh status

# 查看日志
./start_simple.sh logs

# 停止程序
./start_simple.sh stop

# 重启程序
./start_simple.sh restart

# 显示帮助
./start_simple.sh help
```

## 🔧 代理配置（可选）

如果需要使用代理，编辑 `scheduler_config.json` 文件：

```json
{
  "proxy_settings": {
    "enabled": true,
    "http_proxy": "http://127.0.0.1:7890",
    "https_proxy": "http://127.0.0.1:7890"
  }
}
```

支持的代理类型：
- **Clash**: `http://127.0.0.1:7890`
- **V2Ray**: `http://127.0.0.1:1080`
- **Shadowsocks**: `socks5://127.0.0.1:1080`

## 🔍 故障排除

### 1. 程序无法启动
```bash
# 检查环境
./start_simple.sh status

# 查看错误日志
./start_simple.sh logs
```

### 2. OAuth认证问题
如果出现以下错误：
```
❌ 刷新尝试失败: invalid_grant: Token has been expired or revoked
```

**解决方法**：
```bash
# 删除过期的令牌文件
rm -f token.json

# 重新设置OAuth认证
python oauth_setup_v1.0.py

# 重启调度器
./start_simple.sh restart
```

### 3. 网络连接问题
- 检查代理设置是否正确
- 确认代理服务正在运行
- 测试网络连接：`curl -I https://api.binance.com/api/v3/ping`

### 4. 程序意外停止
```bash
# 重新启动
./start_simple.sh restart

# 查看日志了解停止原因
./start_simple.sh logs
```

## 📋 常见问题

**Q: 关闭终端窗口后程序还会运行吗？**
A: 使用 `./start_simple.sh background` 启动的程序会在后台持续运行，关闭终端不影响。

**Q: 关闭Augment和VS Code后程序还会运行吗？**
A: 是的，程序完全独立运行，不依赖任何IDE或编辑器。

**Q: 如何确认程序正在运行？**
A: 运行 `./start_simple.sh status` 查看状态。

**Q: 如何修改执行时间？**
A: 编辑 `simple_scheduler.py` 文件中的 `schedule.every().day.at("07:45")` 部分。

**Q: 程序会自动重启吗？**
A: 程序会持续运行，但如果意外停止需要手动重启。建议定期检查状态。
